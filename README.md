# Parkour
Parkour game made in Python with Ursina Game Engine

Play it here: https://mandaw2014.itch.io/parkour

Join the discord for more news about the game: https://discord.gg/8gVWbs2p

# Controls

WASD - To Move

SPACE - To Jump

ESCAPE - Exit The Game

G - Restart The Level
